# Chris Moore's Fender Code Test

Responsive web test for Fender

### Prerequisites
1. Node
2. NPM
3. Bower
4. Gulp


### Installation
1. Clone
2. npm install
3. bower install


### Usage
1. Compile build with `gulp compile`
2. Run build with `gulp run` - ctrl+c to quit watch process


### Dependencies
1. bootstrap ~3.3.2
1. angular ~1.5.5
3. jQuery 2.1.3
